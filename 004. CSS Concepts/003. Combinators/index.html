Studying about Combinators

Type 1 - Adjacent Sibling Combinator

A+B : All 'B' directly after 'A' are considered

Type 2 - General Sibling Combinator

A~B : All 'B' after 'A' be it direct or not

Type 3 - Child Combinator

A>B : All 'B' nested directly inside 'A'

Type 4 - Descendant Combinator

A B : All 'B' nested inside 'A' direct or not
